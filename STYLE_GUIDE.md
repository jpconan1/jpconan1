# Project Style Guide

## Colors

| Name | Hex | Usage |
|------|-----|-------|
| **Dark Green** | `#01301e` | Background Color |
| **Less Dark Green** | `#2a5733` | Secondary Background / Accents |
| **Pop o Orange** | `#ea461d` | Highlights / CTAs |
| **Good Old Blue** | `#1c50eb` | Links / Primary Actions |
| **Computer Cream** | `#eef1db` | Main Text Color |

## Typography

### Font Family
**Inter** (Google Fonts)

### Styles
*   **Headers**: Inter Black (900), Large size
*   **Body**: Inter Regular (400), Normal size

## Interaction & Animation
*   **Transitions**: None. All hover effects and state changes should be instant (no `transition` css properties).
